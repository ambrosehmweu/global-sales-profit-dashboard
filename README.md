# Global Sales & Profit Analysis Dashboard

## 📊 Project Overview
This project analyzes global cookie sales data across multiple countries (India, UK, Philippines, Malaysia, USA).  
It tracks **Units Sold, Revenue, Cost, and Profit** over time, providing insights into product performance and profitability.

## 🗂 Dataset
- Source: Excel file (`Global Sales & Profit Analysis Dashboard.xlsx`)
- Columns: Country, Product, Units Sold, Revenue, Cost, Profit, Date
- Products: Chocolate Chip, Fortune Cookie, Oatmeal Raisin, Snickerdoodle, Sugar, White Chocolate Macadamia Nut

## 🚀 Key Insights
- Profitability trends vary across countries and products.
- Chocolate Chip consistently drives high revenue across regions.
- Oatmeal Raisin and White Chocolate Macadamia Nut show strong margins in certain markets.

## 🛠 Tools Used
- **Microsoft Excel**: Data cleaning, calculations, dashboard creation
- **Pivot Tables & Charts**: For visualization
- **Profit Analysis**: Revenue - Cost

## 📈 Dashboard Features
- Country-wise breakdown of sales and profit
- Product-level comparison
- Monthly trend analysis
